Amazon Monitoring API Documentation

This API provides the latest, non-cached data about all Amazon products given an ASIN (Amazon Standard Identification Number).

Due to the nature of how data is updated serverside, response times can range from a few ms to ~800ms, depending on Amazon's rate limits.


Available Endpoints

Product Endpoint (GET)

URL: https://amazon.cloudsolve.dev/product/{ASIN}

Query Parameter: key (your designated API key)

Example URL: https://amazon.cloudsolve.dev/product/B07DQSC9B5?key=API_KEY

Usage Endpoint (GET)

URL: https://amazon.cloudsolve.dev/usage

Query Parameter: key (your designated API key)

Example URL: https://amazon.cloudsolve.dev/usage?key=API_KEY

Successful Response:

Status Code: 200

{
   "key": "API_KEY",
   "requests": 5000
}

Erroneous Responses:

Status Code: 401

Invalid API key.
